Для успешной генерации необходимо, чтобы в корневой папке проекта находились следующее файлы и папки:
- students.xlsx : Список студентов, который обязательно должен содержать колонку Student
- Course : папка, в которой находятся задания, теория и хэдеры для ноутбуков, расположенные по неделям. (см. ниже)
- notebookstructure : папка, в которой содержится описание ноутбуков (см. ниже)

1) Папка Course состоит из папок week{} , где {} принимает значениеот 1 до n - количества недель. 
Внутри каждой папки лежат следующее json файлы:
- head.json : Шапка ноутбуков этой недели. Внутри шапки Вы можете автомотически указать имя студента - для этого в тексте ячейке оставьте место, где вы хотите указать
имя студента обознячте как {}.
- theory.json : Словарь с теорие. Важно, чтобы ключ был номером, прописанной строкой, теории.
- task{}.json : Словарь с {} заданием, в котором содержится некоторое количество вариантов этого задания.

Внутри каждый из json файлов должен содержать следующую структуру:

"KEY": [["CELL TEXT","CELL TYPE","CELL NBGRADER TYPE"],...] 
Где: KEY - может быть любым (кроме theory.json)
CELL TEXT - текст/код, который мы хотим видеть для ввода команд Latex, надо указывать косую черту дважды
CELL TYPE - Может принимать два значения: "c"/"m" - код и маркдаун соотвественно
CELL NBGRADER TYPR - Тип метаданных ячейки для nbgradera: NONE/READONLY/AUTOGRADEDTEST/...

2) В папке notebookstructure лежат week{}.txt файлы, которые описывают шаблон ноутубков на {} неделе. В них свой простой язык:

Head 
Th 1 1
Task 1 1

Первая строка говорит нам добавить шапку (шапка будет браться из {} неделе - по названию файла)
Вторая срока говорит взять 1 теорию 1 неделе
Третья строка говорит взять нам 1 задание 1 неделе

При соблюдение всех условий можно спокойно запускать файл main. В итоге из важных для дальнейшего исполтзования файлов будет: папка
NoteBooks с загатовками для nbgradera и файл StudentsWithVariants.xlsx с распределением вариантов задания среди студентов.
